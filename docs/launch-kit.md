I built OpenDesign Vibe Studio: a Windows desktop app for design-first vibe coding.

Good fit for people using Ollama, LM Studio, FreeLLMAPI, LiteLLM, OpenRouter, OpenAI-compatible gateways, or custom local model routers.

## Hacker News / Reddit Style

Show HN: OpenDesign Vibe Studio - local Windows GUI for design-first vibe coding

I wanted a workflow where the design proposal happens before code generation. This is a local Electron app that creates an OpenDesign-style UI proposal, lets you configure custom OpenAI-compatible endpoints, then writes the generated app files into a local project folder with preview.

It is early, Windows-first, and built for people experimenting with local or custom LLM routing.
